# Zenoh

Zenoh is a distributed communication protocol and runtime for data in motion, data at rest, and computations over named resources. It combines publish/subscribe, query/reply, and distributed storage behind key expressions, so applications can route streams, request current values, or access stored samples through the same namespace.

Nodes can run as clients, peers, or routers and form routed or peer-to-peer topologies across LAN, WAN, and constrained links. Zenoh supports multiple transports, discovery mechanisms, and optional integrations such as DDS bridging, with a focus on efficient routing, low overhead, and operation across intermittently connected systems.
